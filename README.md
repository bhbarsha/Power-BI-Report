📊 Power BI Sales Dashboard
🔍 Overview

This project is a sales analytics dashboard built using Power BI. It provides insights into sales performance, YoY growth, customer trends, and regional distribution.

📌 Key Features
KPI Cards (Sales, Orders, Customers)
Year-over-Year (YoY) Analysis
Sales by Category & Region
Interactive Map Visualization
Top Customers Analysis
🛠️ Tools Used
Power BI
DAX
Data Modeling
